Tijdlijn Programmeertalen

Swift 2014

Swift is een programmeertaal die door Apple is ontwikkeld.

Rust 2010

Rust is een programmeertaal die door Mozilla is ontwikkeld. Rust focust op de veiligheid van het programmeren.

Ruby 1995

Ruby is een programmeertaal die in 1995 uitgebracht werd door Yukihiro Matsumoto. De taal werd ontworpen om snel en makkelijk objectgeoriënteerd te programmeren.

Javascript 1995

Javascript is een scripttaal. Die bestaat uit een client-side en server-side. Bij de client-side wordt het uitgevoerd via de browser en bij de server-side via de server.

CSS 1994

Cascading Style Sheets werd in 1994 bedacht door Håkon Wium Lie. CSS is bedoelt voor de opmaak van de HTML pagina's.

PHP 1994

PHP is een scripttaal die in 1994 werd ontworpen door Rasmus Lerdorf. PHP wordt gebruikt om op servers dynamische webpagina's te creëren

HTML 1991

HTML werd in 1991 bedacht en ontwikkeld door Tim Berners-Lee. HTML is een opmaaktaal voor documenten. Het is ook de standaar opmaaktaal voor webpagina's.

Python 1991

Python is een programmeertaal die in 1991 bedacht werd door Guido van Rossum. De bedoeling van Python was om het zogenoemde "saaie werk te automatiseren".
